### 커밋명령어
    - feat : 새로운 기능 추가
    - style : 코드 스타일 수정
    - design : CSS 등 디자인 추가 / 수정
    - comment : 주석 추가 / 수정
    - test : 테스트 추가 / 수정
    - rename : 파일 및 폴더 명 수정
    - remove : 파일 삭제

### branch 명
    - feature/* : 기능 개발
    - test/* : 테스트 추가
